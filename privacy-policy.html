Privacy Policy

Last updated: June 23, 2025

Simple Summary: Element Copy does not collect, store, or transmit any personal data. It only copies HTML elements locally to your clipboard when you request it.

1. Information We Collect

We collect NO personal information. The Element Copy extension:

  • Does not track your browsing activity
  • Does not collect personal data
  • Does not store copied HTML elements
  • Does not transmit any data to external servers
  • Does not use analytics or tracking services

2. How the Extension Works

Element Copy operates entirely locally on your device:

  • When you right-click an element and select "Copy Element", the HTML is copied directly to your system clipboard
  • No data leaves your browser or device
  • The extension only accesses the webpage you're currently viewing when you explicitly use the copy function

3. Permissions Used

The extension requires these permissions solely for functionality:

  • contextMenus: To add the "Copy Element" option to your right-click menu
  • activeTab: To access the current webpage's HTML elements when you choose to copy them
  • scripting: To inject the copy functionality into webpages

4. Data Storage

  • No data is stored by this extension
  • Copied HTML elements are only temporarily placed in your system clipboard
  • The extension does not create any local files or databases

5. Third-Party Services

This extension does not:

  • Connect to any external servers
  • Use third-party analytics
  • Include external scripts or libraries
  • Share data with any third parties

6. Children's Privacy

This extension does not collect any information from anyone, including children under 13. No personal information is processed or stored.

7. Changes to This Policy

If we make changes to this privacy policy, we will update the "Last updated" date. Since the extension doesn't collect data, significant policy changes are unlikely.

8. Contact Information

If you have questions about this privacy policy or the extension, you can:

  • Create an issue on the GitHub repository
  • Contact through the Chrome Web Store support page
Bottom Line: Element Copy is designed with privacy in mind. It performs its function entirely locally and does not collect, store, or transmit any personal information.
